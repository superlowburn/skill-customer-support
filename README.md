# Customer Support Skill

A free agent skill for triaging support tickets, drafting responses, and detecting churn risk.

## Install

```bash
npx skills add automadic/skill-customer-support
```

## What It Does

- Classify support tickets by category, priority, and sentiment
- Draft professional, empathetic responses calibrated to issue type
- Prioritize ticket queues by urgency and customer value
- Flag churn risk customers before they cancel

## Usage

```
use customer-support skill to classify and draft a response to this ticket:
[paste customer message]
```

```
use customer-support skill to prioritize these 10 tickets and flag any churn risks:
[paste ticket list]
```
